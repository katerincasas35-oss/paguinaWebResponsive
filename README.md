Página Web Responsive — Cajas de Colores

Este proyecto es una página web responsive construida con HTML5, CSS3 y Bootstrap 5, cuyo objetivo es practicar y demostrar el uso del sistema de breakpoints y la malla de columnas de Bootstrap.
La página organiza una serie de cajas de colores con diferentes tamaños que se ajustan dinámicamente dependiendo del tamaño de pantalla.

Diseño totalmente responsive

✔️ Uso de Bootstrap 5 y su sistema grid (row, col, breakpoints)

✔️ Más de 15 bloques de colores con alturas personalizadas

✔️ Comportamientos diferentes para pantallas xxl, xl, lg, md, sm y xs

✔️ Código simple, organizado y fácil de entender

 Tecnologías utilizadas

HTML5

CSS3 (estilos internos)

Bootstrap 5.3.8 (CDN)

 Estructura del proyecto

El diseño consiste en una cuadrícula de cajas (divs) con clases personalizadas:

cUno, cDos, cTres, ... cQuince → cada una representa un bloque de color con su propia altura y color.

Cada columna usa clases como:

col-xxl-3  
col-xl-3  
col-lg-6  
col-md-3  
col-sm-6  
col-6


para adaptarse a cada breakpoint.
